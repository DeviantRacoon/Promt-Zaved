# Prompt Saver

Prompt Saver es una aplicacion web para almacenar y administrar plantillas de prompts.

## Desarrollo

1. Instalar dependencias:
   ```bash
   pnpm install
   ```
2. Ejecutar el servidor de desarrollo:
   ```bash
   pnpm dev
   ```
3. Compilar para produccion:
   ```bash
   pnpm build
   ```

Consulta [ARCHITECTURE.md](ARCHITECTURE.md) para detalles sobre la estructura del proyecto.
